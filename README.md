Freeze – Ice Cream & Beverage Landing Page

A responsive single-page website built with pure HTML, CSS, and JavaScript (no frameworks or libraries) to practice front-end fundamentals. The project simulates a fictional ice cream/beverage shop with sections for a hero landing page, menu showcase, location map, and sponsors.

Key features:

- Fully responsive layout with breakpoints for tablet and mobile
- Custom hamburger menu with animated icon transition (CSS transforms + vanilla JS toggle)
- Smooth scroll navigation between sections
- Embedded Google Maps iframe for location display
- CSS Grid/Flexbox for menu card layouts with hover effects
- SVG wave divider for visual transitions between sections

Tech stack: HTML5, CSS3 (Flexbox, media queries, transitions), Vanilla JavaScript (DOM manipulation, event listeners)

Purpose: Built to strengthen core web fundamentals — semantic HTML structure, responsive CSS design patterns, and JS-based interactivity — without relying on frameworks.
